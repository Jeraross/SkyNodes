Puzzle de Grafo — Interior do Aeroporto de Recife

Como o jogador vê e interage com os nós do subgrafo local? Escolha o layout que melhor combina com a estética Atari 2600 do GDD.

AEROPORTO RECIFE — SUBGRAFO
► Arraste para criar arestas entre os nós

A — Nós como caixas rotuladas

Vértices são retângulos com nome do setor. Conexões são linhas entre eles. Clique num nó e arraste para o outro para criar aresta. Verde = conectado, vermelho = corrompido, cinza = inativo.

AEROPORTO RECIFE — SUBGRAFO
► Clique em dois nós para conectá-los

B — Nós como círculos (clássico de grafo)

Vértices são círculos dispostos radialmente, hub central mais visível. Visual mais próximo de diagramas de grafos tradicionais. Clique em dois círculos para criar aresta. Mais didático para o conceito de teoria dos grafos.

AEROPORTO RECIFE — SUBGRAFO
► Clique num nó depois em outro para conectar

C — Nós como ícones de setor (pixel art)

Cada setor tem um ícone representativo além do nome. Mais imersivo e narrativo. Visualmente mais rico mas potencialmente mais poluído. Boa âncora para o jogador entender o que cada nó representa na história.
